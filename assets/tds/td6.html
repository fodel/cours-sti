1) Que permet la propriété opacity en CSS ?

Correction

La propriété opacity en CSS permet d'ajuster la transparence d'un élément et de le cacher visuellement partiellement. Elle ne contrôle pas le débordement d'un conteneur.

2) Laquelle(s) de ces instructions vérifient si un caractère x existe une seule fois dans la chaine ch :

Correction

La méthode ch.indexOf(x)==ch.lastIndexOf(x) vérifie si un caractère x existe une seule fois dans la chaîne ch. Si l'index de la première occurrence est égal à l'index de la dernière occurrence, cela signifie que le caractère n'apparaît qu'une seule fois.

3) Lesquel(s) de ces événements se déclenchent lorsqu'on modifie une zone de texte ?

Correction

Les événements Oninput et Onkeyup se déclenchent lorsqu'on modifie une zone de texte. L'événement Onmouseover se déclenche lorsque le curseur de la souris passe sur un élément.

4) Quels sont les rôles respectifs des balises <thead> et <tbody> dans un tableau HTML ?

Correction

La balise <thead> est utilisée pour regrouper les en-têtes de colonnes d'un tableau, tandis que la balise <tbody> est utilisée pour regrouper le corps des données d'un tableau.

5) Quel est le rôle de l'attribut placeholder dans un champ de formulaire HTML ?

Correction

L'attribut placeholder affiche un texte indicatif à l'intérieur d'un champ de formulaire qui disparaît dès que l'utilisateur commence à taper. Il n'envoie pas de valeur par défaut au serveur.

6) Que permet de faire la propriété CSS suivante lorsqu'elle est appliquée à une liste ?
list-style-image: url("img.png");

Correction

La propriété list-style-image permet d'utiliser une image comme marqueur pour les éléments d'une liste. Elle remplace les puces par défaut par l'image spécifiée.

7) Que retourne les instructions suivantes, en langage JavaScript ?
var ch = "Bac"; var code = ch.charCodeAt(0); alert(code) ;

Correction

La méthode charCodeAt(0) retourne le code Unicode du premier caractère de la chaîne "Bac". Le code Unicode de 'B' est 66.

8) Que retourne l'instruction parseInt("20px")+20, en langage JavaScript ?

Correction

La fonction parseInt("20px") extrait le nombre entier du début de la chaîne, ce qui donne 20. Ensuite, 20 + 20 = 40.

9) Que retourne l'instruction strcmp("bac", "bac") en langage PHP ?

Correction

La fonction strcmp() en PHP compare deux chaînes de caractères. Elle retourne 0 si les chaînes sont identiques, une valeur négative si la première chaîne est inférieure à la seconde, et une valeur positive si la première chaîne est supérieure à la seconde. Comme "bac" et "bac" sont identiques, la fonction retourne 0.

10) Quelle propriété CSS permet de désaturer une image et de la rendre en niveaux de gris ?

Correction

La propriété CSS filter: grayscale(100%) permet de désaturer une image et de la rendre entièrement en niveaux de gris.
